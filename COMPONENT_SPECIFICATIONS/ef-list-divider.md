# ef-list-divider

## Description

An `<li>` element serving as a separator between other `<li>` elements.


## Influences

* http://getuikit.com/docs/list.html
* http://semantic-ui.com/elements/list.html

* None

* **Block support:** *No*
* **Non-Block support:** *Yes*

```
// template

{{ef-list-divider}}

// rendered HTML

<li class="ef-list-divider"></li>
```
